# Talky Chef

Рецепты с голосовым помощником.
Перемещайтесь по рецепту, даже если руки заняты!

talkychef.ru

Мой первый проект на Flutter, успел поработать с

- firestore database

- rive animations
- navigator 2.0

В данный момент проект собирается под Android и Web.
